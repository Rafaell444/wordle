# wordle
ეს არის Wordle-თამაშის მინი პროტოტიპი დაწერილი Python-ზე.
## Installation
გადმოიწერეთ ფაილები სათამაშოდ.

ასევე დააინსტალირეთ პითონის მინიმუმ მე-3 ვერსია.

## Usage
 შედით გადმოწერილ ფაილში:
 
```bash
cd/ფაილი
```

შემდეგ გაუშვით main.py ფაილი პითონით:

```bash
python3 main.py
```
